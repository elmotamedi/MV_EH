# MV_EH dataset
This repository contains a dataset capturing the eudaimonic and hedonic qualities of users and their favourite music videos across various genres. It includes user-generated ratings and annotations obtained through a user study and expert annotations. The dataset comprises 1535 interaction data points from 88 users. Music video experts annotated the music videos, while the users' ratings and eudaimonic and hedonic orientations were collected through a user study.
